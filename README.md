# PrimeLink Global Services

Premium website for **PrimeLink Global Services** — a product sourcing & global logistics company based in India.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Font**: Manrope (Google Fonts)
- **Deployment**: Vercel

## Pages

| Route | Description |
|-------|-------------|
| `/` | Home |
| `/about` | About Us |
| `/services` | Services Index |
| `/services/[slug]` | Dynamic Service Pages (9 services) |
| `/how-it-works` | 10-Step Process |
| `/faq` | FAQ |
| `/contact` | Contact |

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy to Vercel

Push this repo to GitHub, then import it at [vercel.com/new](https://vercel.com/new). No environment variables required.
